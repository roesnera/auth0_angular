What does this app do?

This app uses Auth0 to authenticate/authorize users

Auth0 is an authorization server solution that has the ability to connect with many different common authentication providers. It will play the role of our federated authentication server.

When a user is authenticated, they can access more api endpoints

Once a user is signed in, they gain access to new pages and when those pages request resources from our api, their credentials are checked. Our resource server will validate the credentials with Auth0, as is standard with access code flow.

Auth0 provides permissions (AKA scopes and roles) using rules

Not all users are equal in their access to protected data. Using JavaScript, we can include scope in Auth0 provided tokens for our resource server to decode, providing further protection for sensitive data.
